GitNote is a powerful note-taking app that combines the simplicity of Markdown with the power of Git version control.

## Key Features

- **Markdown Support**: Write notes in Markdown with full syntax highlighting
- **Git Integration**: Sync your notes across devices with automatic commits
- **Task Management**: Convert notes to tasks with completion tracking
- **Frontmatter Metadata**: Organize notes with YAML headers

## Getting Started

1. Clone a Git repository
2. Create your first note
3. Start writing in Markdown
4. Sync changes with Git

> **Tip**: Use frontmatter to add metadata like titles, tags, and completion status to your notes.
